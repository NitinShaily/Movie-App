# Movie-App  
## Description
A Web App to visualize the top-rated movies.

## Functionalities
> 1.  Quick Search to sort movies on the basis of need  
> 2. Description of the movie on the hover of the mouse over movie template  

## Technolgies used
> HTML  
> CSS  
> JavaScript
